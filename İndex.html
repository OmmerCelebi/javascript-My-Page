A Short History of React

Facebook created React.js for its own use in 2011. In 2012, it was used on Instagram, a subsidiary of Facebook. In 2013, Facebook made React.js open source. In 2014, many large companies started using React.js in production environments. In 2015, Facebook also open sourced React Native (React Native is a library that allows us to build native mobile apps on Android and iOS using React.js). In 2016, with version 15, React.js started using Semantic Versioning. This also helped communicate to the developer community that React is more stable. Today React.js is used by many Fortune 500 companies. Facebook has a full-time React development team. They regularly release bug fixes, enhancements, blog posts and documentation.
